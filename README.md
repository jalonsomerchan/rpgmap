# JS Template

Template moderno para proyectos frontend con HTML, JavaScript y Tailwind CSS.

## Stack

- Vite
- Tailwind CSS v4
- ESLint
- Prettier
- GitHub Actions

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Formatear código

```bash
npm run format
```

## Estructura

```txt
src/
├── assets/
├── css/
└── js/
```
